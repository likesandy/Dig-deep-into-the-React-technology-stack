- 组件的名称问题:在 ES6 中类表达式中类名是可以省略的
- 组件的名称都可以通过 displayName 来修改
- 高阶组件并不是 React API 的一部分,它是基于 React1 的组件特性而形成的设计模式
- 高阶组件在一些 React 第三方库中非常常见
  - 比如 redux 中的 connect
  - 比如 react-router 中的 withRouter
